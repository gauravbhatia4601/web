# Learn How To Code #

Introduction

C is a high-level and general purpose programming language that is used for developing firmware or portable applications. C was developed at Bell Labs by Dennis Ritchie for the Unix Operating System in the early 1970s. In 1978, Brian Kernighan and Dennis Ritchie produced the first publicly available description of C, now which is known as the K&R standard.

Facts About C

● Basically the C was invented to write an operating system named as UNIX.
● C is a successor language of B language which was invented around the early 1970´s.
● After C was formalized by the American National Standard Institute(ANSI).
● The UNIX operating system was totally written in the C language.
● Now the C is widely used and popular system programming language.
● Today´s most popular Linux OS and RDBMS My SQL are written in the C.
